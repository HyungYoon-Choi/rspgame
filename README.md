# 가위, 바위, 보!!!

이 프로젝트는 HTML, CSS, 그리고 JavaScript를 사용하여 만든 간단한 가위바위보 게임입니다.

## 기능

- 컴퓨터와 가위바위보 게임 플레이
- 애니메이션 캐릭터
- 반응형 디자인
